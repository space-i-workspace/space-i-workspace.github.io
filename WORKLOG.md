# 공간i 비상주사무실 웹사이트 작업기록

## 개요
강남 서초 하이브랜드 빌딩 비상주사무실 "공간i"의 자사 랜딩페이지.
기존 카페24 쇼핑몰 대신 독립 정적 사이트로 제작.

## 아키텍처
- 원페이지 랜딩 (수직 스크롤, 9개 섹션)
- HTML + Tailwind CSS(CDN) + Vanilla JS
- 빌드 도구 없음, 정적 호스팅 가능

## 페이지 구성

### index.html (메인 원페이지)
1. 고정 네비게이션 (스크롤 블러 효과)
2. 히어로 (메인 카피 + CTA)
3. 신뢰 배지 바 (브로드컴, 삼성 에스원, 세브란스)
4. 3단계 신청 프로세스
5. 차별점 6가지 (카드 그리드)
6. 요금 안내 (개인/법인/프리미엄)
7. 공간 안내 (이미지 갤러리 + 지도)
8. 경쟁사 비교표
9. FAQ (아코디언)
10. CTA + 푸터

### product.html (상세 상품 페이지)
- 풀스크린 히어로 + 공간 스토리 + 주문/결제

### why-virtual-office.html (왜 비상주사무실인가)
- 카페24 상세페이지 자료(detail/ 이미지) 활용
- 구성: 폐업 위험 경고 → 건물주 직영 비교 → 상주매니저 → 후기 → 입주기업 → 공간투어 → 부가서비스 → CTA
